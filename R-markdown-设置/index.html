使用Rmarkdown来写报告

Rstudio在之前的版本中早已支持Rmarkdown来进行报告的输出，可根据情况来选择输出Html,pdf,word的报告形式。这篇博客主要是用来记录一下怎么设置Rmarkdown，具体的使用详见官网R markdown。

Rmarkdown设置pdf

Rmarkdown使用Tex引擎来生成pdf，因此，需要在Rmarkdown中控制Tex的参数，其中有：

目录：

如果不设定toc_depth，默认是3，即3级目录

图片设置：

  • fig_width：图片的长
  • fig_height：图片的宽度
  • fig_corp：默认为真
  • fig_caption：图片是否配说明文字
  • dev：默认为真

表格打印

  • df_print：default，kable，tibble

语法高亮

  • highlight：default，tango，pygment，kate，monochrome, espresso, zenburn, haddock，null（语法不高亮）

Latex选项

  • lang：
  • fontsize: 字体的大小
  • documentclass: 文档的类型
  • classoption: 文档类型的设置
  • geometry: 文档边界的设置
  • mainfont,sansfont,monofont,mathfont：文档字体的选择
  • linkcolor,urlcolor,citecolor: 文档中链接的颜色（red,green,magneta,cyan,blue,black）

Latex引用Package选项

高级设置

  • Latex引擎
