Barra风险模型的细节

发表于 2018-05-01 | 分类于 金融
字数统计 1,374

从Barra CNE5手册中我们可以看到中国模型的主要亮点与美国模型类似，主要是一些细节的处理以及加入长期和短期风险模型的管理，下面简单介绍一下Barra的处理细节

特征值调整的样本协方差（Optimization Bias Adjustment）

这个改进是基于由于历史数据估计样本协方差矩阵存在估计误差的问题，其主要使用Menchero,Wang,Orr(2011)的方法，调整样本协方差。主要步骤如下：

  1. 首先计算出因子收益率样本协方差$V_0 = \frac{f’f}{n-1}$
  2. 计算样本协方差矩阵的特征值和特征向量$D_0 = U_0’V_0U_0$
  3. 使用正态分布模拟出因子收益率$f’_m = U_0b_m$
  4. 使用模拟出的因子收益率$f’_m$计算模拟出的样本协方差矩阵$V_m$
  5. 计算模拟出的样本协方差矩阵的特征值和特征向量$D_m = U_m’V_mU_m$
  6. 使用模拟出的特征向量乘上原始协方差矩阵得到“真实”的特征值矩阵$D_r = U_m’V_0U_m$
  7. 计算每一个模拟特征值与“真实”特征值的比例系数$$\lambda(k) = \frac{1}{m}\sqrt{\frac{D_r(k)}{D_m(k)}}$$
  8. 构建放缩系数$$\gamma(k) = a(\lambda(k)-1)+1$$
  9. 调整$D = \gamma^2D_0$
  10. 计算$V_0 = U_0DU_0’$

通常$a = 1.4$
这种方法的基本思想就是利用协方差矩阵的特征因子模拟不同样本下的样本协方差矩阵，并对这些样本协方差矩阵求均值，以此来减少样本协方差矩阵在估计上的估计误差。

风险模型滞后效应的调整(Volatility Regime Adjustment)

这个改进在于对于长期风险模型来说，短期的风险变化往往不能够在数据中立刻显现出来，表现为风险估计往往具有滞后性，为解决这一问题，Barra提出了一种解决方案如下：

  1. 基于Bias Statistic统计量的风险预测能力检验在因子收益率波动上仍然成立
  2. 计算不同因子在$t$时刻的Bias Statistic统计量（使用$t$时刻已实现的因子收益率和$t-1$时刻预测的因子波动率）$b_kt$，与Bias Statistic统计量类似，在截面上应该标准差接近于1，$B_F = \sqrt{\frac{1}{n}\sum_{k=1}^n{b_kt^2}}$
  3. 再在时间上去对$B_F$的均值作为调整系数，$\lambda = \sqrt{\sum_{t=1}^T{B_{Ft}^2w_t}}$，$w_t$表示时间权重，往往使用指数加权系数。
  4. 对$t$时刻的风险重新进行调整为$\sigma = \lambda\sigma_0$

从方法本身来看，对短期风险调整的方式可以看作是使用上期风险预测能力的表现（Bias Statistic）对下期风险进行相应的调整，上期的Bias Statistic小了说明上期风险估计过高，则下期相应的调小风险估计，反之亦然。只能说这样的方式可能会缓解风险出现重大变化的情形，例如股灾。

特异风险调整（Specific Risk Model with Bayesian Shrinkage）

特异风险调整的目的也是最近思考的来的，这个还真是只有亲手去实践才能发现其中的奥妙。

特异风险数据缺失调整

在实务中，我们构建结构化风险模型，常常会遇到某些特殊的股票样本，例如长期停牌，刚上市不久，由于这类股票的样本性质不佳，往往在构建因子模型的时候会将这类股票剔除样本。这一步本身并没有问题，问题在于对于投资组合来说，这类样本往往也会成为投资标的，也需要进行风险管理，例如上市不久的股票，但是这类股票的特异风险的估计成为一个问题，一个简单的做法是将这类股票的特异风险估计为整体特异风险平均数的某一倍数。Barra提出了另一种比较复杂的解决方案：

  1. 首先计算所有可以计算的股票的特异风险$\sigma_n = C_n^{nw}[\sum_t{w_t(u_t-\overline{u}_t)}]$，其中$C_n^{nw}$是NeweyWest自相关调整系数，$w_t$是时间加权系数，通常是指数加权。
  2. 对特异风险构建结构化模型，这里选择的结构化因子则是需要所有股票均有的特征也就是说所有股票都有的指标，不管其本身样本如何。
  3. $ln(\sigma^{TS}) = \sum{Xb}+\varepsilon)$
  4. 最终计算的特异风险为$\sigma = \gamma_n\sigma_n + (1-\gamma_n)\sigma_n^{TS}$

特异风险贝叶斯缩减

在Barra的研究中发现，特异风险的Bias Statistic估计量与特异风险紧密相关，具体点来说，是随着特异波动的增大，对应的Bias Statistic也会增大。为减轻这一效应，Barra提出下面这一解决方案：

  1. 首先将所有股票按照残差波动率的分位点分为$k$个组
  2. 分别计算每个组内的残差波动率的均值$\overline{\sigma}(s_k) = \sum_{i=1}^{s_k}{w_i\sigma_i}$，其中$w_i$是每只股票在每个组内的市值权重。
  3. 缩减后的残差波动率为$\sigma^{SH} = \upsilon_n\sigma_{s_k} + (1 - \upsilon_n)\sigma_n$
  4. $\upsilon_n = \frac{q|\sigma_n - \sigma_{s_k}|}{std_{s_k}(\sigma_n)-|\sigma_n - \sigma_{s_k}|}$，$q$是经验缩减系数
